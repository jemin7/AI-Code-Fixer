# AI Code Fixer 🚀

A powerful, full-stack web application that uses Google's Gemini AI to identify bugs and provide instant code fixes. Whether you're working on a single script or a multi-file web project, this tool streamlines the debugging process.

## 🔗 Live Demo
Check out the live application here: [AI Code Fixer](https://ai-code-fixer-azure.vercel.app/)

## ✨ Features

- **Single File Mode:** Support for JavaScript, TypeScript, Python, Java, and C++.
- **Web Project Mode:** Analyze HTML, CSS, and JavaScript simultaneously to see how they interact and fix cross-file bugs.
- **VS Code Engine:** Powered by the Monaco Editor for familiar syntax highlighting and IntelliSense-style autocomplete.
- **Smart "No Bug" Detection:** If your code is already perfect, the AI will let you know instead of rewriting it.
- **One-Click Copy:** Quickly grab the fixed code with a dedicated copy button.
- **Persistent Sessions:** Your code is saved to local storage, so you won't lose progress if you refresh the page.

## 🛠️ Tech Stack

- **Frontend:** React, Vite, Tailwind CSS, Monaco Editor, Axios.
- **Backend:** Node.js, Express, TypeScript.
- **AI:** Google Generative AI (Gemini 1.5 Flash).
- **Deployment:** Vercel (Frontend) & Render (Backend).
